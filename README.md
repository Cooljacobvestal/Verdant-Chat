# Verdant Chat

A sleek, modern chat UI prototype inspired by Material You design principles.

## Highlights

- Glassmorphism + tonal surfaces with light/dark mode.
- Responsive layout with conversation list + active chat panel.
- Familiar messaging UX patterns (typing state, reactions, quick actions, composer).
- Lightweight vanilla HTML/CSS/JS so it runs anywhere.

## Run locally

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
